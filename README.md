uDonate is a web-based donation platform prototype built to simplify the process of connecting donors with people and causes in need. The project focuses on a clean user interface and smooth donation flow.

Features
	•	Simple and intuitive donation interface
	•	Displays donation campaigns and causes
	•	Responsive and user-friendly design
	•	Functional frontend prototype

Tech Stack
	•	TypeScript
	•	JavaScript
	•	CSS

Project Type

This is a working frontend prototype developed during an E-Summit event to demonstrate a donation platform concept.

Use Case

uDonate highlights how web technologies can be used to build accessible platforms for social impact.
